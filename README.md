# Site de apresentação de Trabalho de Conclusão de Curso

### Descrição

Esse é um site criado para o Trabalho de Conclusão do Curso de Ciência da Computação na Universidade Federal de Minas Gerais.

O tema do projeto é `Análise da Presença Feminina em Cursos de Computação na Universidade Federal de Minas Gerais`.

Após realização de uma pesquisa com alunas, os resultados foram apresentados nesse site.

Ele pode ser visualizado online por esse [link](https://mulheres-computacao-ufmg.netlify.app/) .
